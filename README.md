## NodeJS Express with Babel
```
Create Basic node-express project with basic route
```

### Setup guide
1. npm run setup
	- setup .env file
2. npm install
	- Install plugins
3. npm start
	- Run Project


### Project branchs
1. master
	- base code
2. dev
	- development code
3. connect-sequelize [ WIP ]
	- Api with sequelize connection
4. connect-mongoo [ Pending ]
	- Api with mongodb databse.
